# Darshan Ease Project

## Project Architecture

### Project Overview
This project is developed as part of my internship. It demonstrates a simple web application with basic functionality and structured code.

## Structure
- client: Frontend
- server: Backend

## Frontend
- Built using: HTML / CSS / JavaScript
- Purpose: Handles user interface and user interactions

## Backend
- Built using: Node.js / Express
- Purpose: Handles business logic and API requests

## Database
- Database used: MongoDB

## Architecture Flow
User → Frontend → Backend API → Database → Response → Frontend UI
