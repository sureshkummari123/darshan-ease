# DarshanEase Project

## Project Overview

DarshanEase is a full-stack MERN web application developed as part of my internship. The application helps users explore temple darshan slots and book tickets online. It provides a user-friendly interface for booking management and uses modern web technologies.

### Structure

- client: Frontend React application
- server: Backend Node.js and Express.js application

### Frontend

**Built using:**
- React.js (Vite)
- HTML
- CSS
- JavaScript

**Purpose:**
Handles the user interface, temple details display, booking form, and user interactions.

### Backend

**Built using:**
- Node.js
- Express.js

**Purpose:**
Handles API requests, booking operations, and communication between frontend and database.

### Database

**Database used:**
- MongoDB

**Purpose:**
Stores and manages booking information.

## Architecture Flow

User → React Frontend → Express API → MongoDB Database → Response → Frontend UI
